## Sobre el Proyecto

**FIRSTHub Web** es la cara pública del ecosistema FIRSTHub.

Su propósito es ofrecer una presencia profesional y moderna para mostrar la misión, recursos y valor de la comunidad FIRST, especialmente para equipos de habla hispana.

No es solo una página informativa.  
Es una plataforma de crecimiento para equipos STEM.

---

## Objetivos Principales

- Dar visibilidad al proyecto FIRSTHub.
- Inspirar nuevos estudiantes a entrar a robótica.
- Mostrar recursos disponibles para equipos.
- Centralizar acceso a documentación y herramientas.
- Generar una imagen profesional del ecosistema.
- Facilitar contacto con escuelas, patrocinadores y comunidad.
